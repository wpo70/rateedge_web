07 / Usage Guidelines

Do's and Don'ts

✓ Do

  • ✓ Use approved color combinations
  • ✓ Maintain clear space around logo
  • ✓ Use on solid backgrounds
  • ✓ Scale proportionally
  • ✓ Use dark version on dark backgrounds

✗ Don't

  • ✗ Change the colors
  • ✗ Distort or stretch
  • ✗ Add shadows or effects
  • ✗ Rotate the logo
  • ✗ Use on busy backgrounds

Minimum Sizes

  • Digital: 120px width minimum
  • Print: 30mm width minimum
  • Icon: 24px minimum
  • Favicon: 16px minimum
